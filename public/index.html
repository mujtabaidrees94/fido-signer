WebAuthn Demo

This application demonstrates WebAuthn for key generation and data signing.

1. Register a FIDO Key

Create a new WebAuthn credential that will be used for authentication and signing.

2. Authenticate with Your Key

Verify your identity using the registered WebAuthn credential.

3. Sign Data with Your Key

Use your WebAuthn credential to cryptographically sign arbitrary data.

4. Verification Details

Technical details about the signature verification.
